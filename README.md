# WDDM130 Lab 2 - Ticket Booth Validator

Files:
- index.html
- style.css
- script.js

How to run:
1. Open index.html in your browser
2. Enter ticket type (REG / STU / VIP), quantity, and optional promo SAVE5
3. Click Purchase to add a receipt
4. Click Finish / Summary to view totals

Notes:
- Validation errors use alert() (per class note)
- Uses do-while in validation logic
