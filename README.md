Description  
This project implements a client-side order form using HTML, CSS, and JavaScript.  
User input is validated, calculations are performed in JavaScript functions, and results are displayed dynamically in an HTML table.

Features  
- Form input validation (customer name and quantity)  
- Product pricing with discounts and tax calculation  
- Dynamic table row generation for each order  
- Running summary totals (orders, quantity, revenue)  
- Client-side only (no arrays, objects, JSON, or server-side code)

Files  
- index.html – Page structure and form  
- script.js – Validation, calculations, and DOM updates  
- style.css – Basic styling

Setup / Run  
1. Open index.html in a browser  
2. Enter order details and submit  
3. Each valid order is added to the table and summary updates automatically
